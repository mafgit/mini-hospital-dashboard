# Mini Hospital Dashboard

### This project features:
- Fernet Encryption & Decryption
- Hashing
- Masking
- GDPR
- Asking Consent
- Role-Based Access (Doctors, Receptionists, Admins)
- Date Retention Policy
- Action Logging
- Analytic Graph
- Adding/Editing Patients
- Navigation
- Login/Logout

### Made using:
- Streamlit
- SQLite
- Python
